# Philosophy Writing Output Contracts

Use these only when structure helps.

## Drafting Output

```text
Passage function:
Assumptions:
What must be preserved:
Draft:
Why this structure works:
Unresolved risks:
Citation/source checks:
```

## Review Output

```text
Main issue:
Why it matters:
Suggested fix:
Substantive alternative, if any:
Lower-priority issues:
Source or citation risks:
```

## Revision Output

```text
Revised prose:
What was preserved:
Change summary:
Argumentative effect:
Remaining pressure:
```

## Argument Repair Output

```text
Current claim:
Problem:
Repaired argument:
Repair provenance:
Hidden assumption:
Likely objection:
Reply strategy:
```

## Objection-and-Reply Output

```text
Target:
Objection:
Why it has force:
Reply:
Cost of the reply:
Stronger version to consider:
```

## Literature-Review Output

```text
Problem:
Positions:
What each explains:
Residual pressure:
Project location:
Draft prose:
What not to overclaim:
Source checks:
```
